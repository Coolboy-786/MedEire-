# Submission Checklist

The submission should include the project source code plus the following required files.

## Dockerfiles

- `client/Dockerfile`
- `client/Dockerfile.dev`
- `server/Dockerfile`

## Docker Compose

- `docker-compose.yml`
- `docker-compose.local.yml`
- `docker-compose.dev.yml`
- `docker-compose.cicd.yml`

## Deployment Instructions

- `DEPLOYMENT.md`
- `README.md`

## Environment Variable Configuration

- `ENVIRONMENT_VARIABLES.md`
- `server/.env.example`

## Technical Report

- `docs/technical-report.md`

If a PDF is required, open `docs/technical-report.md` in Word, Google Docs, or another Markdown editor and export it as `technical-report.pdf`.
